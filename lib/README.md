# wkhtmltox Libraries

Place the wkhtmltox library files in the appropriate directories:

## 32-bit (win32/)
- `wkhtmltox.lib` - Import library
- `wkhtmltox.dll` - Runtime DLL

## 64-bit (win64/)
- `wkhtmltox.lib` - Import library
- `wkhtmltox.dll` - Runtime DLL

## Download Links
- 32-bit: https://github.com/wkhtmltopdf/packaging/releases/download/0.12.6-1/wkhtmltox-0.12.6-1.msvc2015-win32.exe
- 64-bit: https://github.com/wkhtmltopdf/packaging/releases/download/0.12.6-1/wkhtmltox-0.12.6-1.msvc2015-win64.exe

Extract using 7-Zip to get the lib and bin folders containing the required files.
