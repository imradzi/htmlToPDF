# Templates Directory

Place your letterhead image here as `letterhead.png`.

## Recommended Specifications
- **Format:** PNG (transparent background supported)
- **Width:** ~800px (will scale to fit page width)
- **Height:** ~100-150px
- **Content:** Company logo, name, address, contact info

## Example Letterhead Layout
```
+--------------------------------------------------+
|  [LOGO]   Company Name                           |
|           Address Line 1, City, Country          |
|           Tel: xxx | Email: xxx | Web: xxx       |
+--------------------------------------------------+
```

If no letterhead image is found, the PDF will be generated without it.
